## Purpose
Use this skill to generate **UI/UX design direction** and concrete, production-ready suggestions:
- UI styles, color palettes, typography pairings
- UX guidelines and heuristics
- Component patterns and layout guidance
- Chart / data-viz type suggestions

## How to use (in this repo)
Ask for the output you want and include context:
- Product + audience + goal
- Screen/page (or component) you’re working on
- Constraints (RTL/Hebrew, brand colors, tone, accessibility level)

Examples:
- "תציע 3 כיווני UI למסך הגדרות (RTL), כולל פלטת צבעים, טיפוגרפיה וטון כתיבה."
- "תן ביקורת UX על קומפוננטת ‘אימון ניסיון’ + הצעות לשיפור היררכיה וקוגניטיב לואד."
- "תציע מערכת טוקנים (colors/type/spacing) שתתאים ל-HeyZoe."
